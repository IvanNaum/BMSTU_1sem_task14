# Flask Shop

1. Создать .env:

```
SECRET_KEY="<secret_key>"
```

2. Запустить приложение:

``` 
flask run
```


### Фичи

1. Генерация форм
2. Хэширование паролей
3. Ajax likes

### Реальзовано
1. Cтраница "Добавление товаров"
2. Страница "Карточка товаров"
    * Редактирование 
    * Удаление
    * "Звездочки" AJAX
    * Просмотр коментариев
3. Страница "Каталог товаров"
